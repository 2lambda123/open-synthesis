How to Contribute

The Open Synthesis Project is available open-source under the GNU Public License Version 3 at https://github.com/twschiller/open-synthesis

In addition to intelligence analysis, we're actively looking for the following kinds of contribution:

  • UI/UX design
  • Front-end development
  • Translation
  • Bug Reports

A list of open development tasks is available via GitHub Issues. If there's another way you'd like to contribute, email us at contribute@opensynthesis.org.

Reporting Bugs

Find a bug? The best way to report it is using GitHub Issues. However, you can also email us at bugs@opensynthesis.org.

Responsible Disclosure

If you find a security vulnerability in Open Synthesis, please privately report it to security@opensynthesis.org. For secure communication, please use this PGP key.

Our responsible disclosure program is available via GitHub.

Media Inquiries

Are you a journalist or blogger interested in learning more about the Open Synthesis Project? Send us an email at media@opensynthesis.org.
